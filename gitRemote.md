## Remote - 协作
git remote 
git remote -v
git remote add <name> <url>
git remote rm <name>
git remote rename <old-name> <new-name>

git fetch + git merge = git pull

git fetch 作用:
```
1、从远程仓库下载本地仓库中缺失的提交记录
2、更新远程分支指针(如 origin/master)
```


git pull --rebase <remote> 

git push <remote> <branch>
git push <remote> --all
git push <remote> --tags
